API Документация

🔐 OAuth2 Авторизация

WETMO использует стандартный протокол OAuth2 для безопасной авторизации.

// Шаг 1: Редирект пользователя на страницу авторизации GET /oauth/authorize?client_id=YOUR_CLIENT_ID&redirect_uri=YOUR_REDIRECT_URI // Шаг 2: Обмен кода на access_token POST /oauth/token { "client_id": "YOUR_CLIENT_ID", "client_secret": "YOUR_CLIENT_SECRET", "code": "CODE_FROM_STEP_1" } // Шаг 3: Получение информации о пользователе GET /oauth/userinfo Headers: Authorization: Bearer YOUR_ACCESS_TOKEN

🤖 Bot API

Боты подключаются через WebSocket и могут отправлять/получать сообщения.

// Подключение бота const io = require('socket.io-client'); const socket = io('https://wetmo.app', { auth: { token: 'BOT_TOKEN' } }); // Отправка сообщения socket.emit('send_msg', { target: 'username', message: 'Hello from bot!' }); // Получение сообщений socket.on('receive_msg', (data) => { console.log(data.sender, data.message); });

📡 WebSocket события

// Присоединение к чату socket.emit('join', { chat_id: 'user1-user2' }); // Присоединение к каналу socket.emit('join', { channel_id: 123 }); // Получение сообщений чата socket.on('receive_msg', (data) => { ... }); // Получение сообщений канала socket.on('receive_channel_msg', (data) => { ... });
